Auto AGI Builder Deployment System

This guide demonstrates how to use the Auto AGI Builder deployment system to deploy your application to Vercel with proper domain configuration. The system consists of three main components:

Complete Deployment Process:
Run deploy-production.bat to deploy your application and consolidate domains
Run run-dns-config.bat to configure DNS records for your custom domain
Run verify-domain.bat to verify domain configuration after DNS propagation
1

Full Deployment

The deploy-production.bat script handles the complete deployment process, including domain consolidation, application deployment, and environment variable configuration.

When to use: Run this script first to deploy your application and set up domains in Vercel.

Features:

  • Consolidates domains assigned to incorrect projects
  • Deploys the application to Vercel (from GitHub or local files)
  • Configures environment variables
  • Sets up domain redirects
  • Verifies deployment status

How to Run:

Double-click deploy-production.bat
View Deployment Demo
2

DNS Configuration

The run-dns-config.bat script helps you configure the correct DNS records for your custom domain, providing tailored instructions for your specific DNS provider.

When to use: After deploying, run this script to get detailed DNS configuration instructions for your domain.

Features:

  • Checks current DNS records
  • Detects your DNS provider and generates tailored instructions
  • Creates detailed configuration files with exact DNS records
  • Supports popular DNS providers (Vercel DNS, GoDaddy, Namecheap, Cloudflare, AWS Route 53)

How to Run:

Double-click run-dns-config.bat
View DNS Configuration Demo
3

Domain Verification

The verify-domain.bat script verifies the status of your domains and DNS records to ensure everything is properly configured.

When to use: After DNS propagation (24-48 hours), run this script to verify your domain configuration.

Features:

  • Checks verification status with Vercel
  • Verifies DNS records point to Vercel servers
  • Confirms nameserver configuration
  • Tests domain accessibility
  • Checks SSL certificate validity

How to Run:

Double-click verify-domain.bat
View Domain Verification Demo

Common Issues & Troubleshooting

Domain Issues

  • Domain not verified: Make sure you've added the required TXT records and wait for DNS propagation (24-48 hours)
  • DNS records not updated: Check your DNS provider's dashboard to ensure records were saved correctly
  • Domain assigned to wrong project: Use deploy-production.bat to consolidate domains to the correct project

Deployment Issues

  • Build failures: Check Vercel logs for error messages
  • Missing environment variables: Make sure all required environment variables are set
  • API errors: Ensure your Vercel API token is valid
