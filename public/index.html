LocalLift Documentation

Introduction

Welcome to the LocalLift documentation. This guide provides comprehensive information about the platform, its modules, and how to use them effectively. LocalLift is a modular platform designed to help local businesses improve their online presence and engagement.

Getting Started

To run the LocalLift application locally: 

uvicorn main:app --reload

The application will be available at http://localhost:8000.

Core Modules

Sales Team Module

The Sales Team module provides tools for managing sales operations, tracking performance, and analyzing trends.

Key Features:

  • Sales Rep Dashboard at /admin/sales-dashboard
  • Trend Analysis with top performer tracking
  • Bonus payment scheduling and alerts
  • Regional performance comparison

Client Module

Tools for client interaction, engagement tracking, and reporting.

Admin Module

Administrative tools for platform management and oversight.

Deployment

Vercel Deployment

vercel --prod --confirm

Railway Deployment

# Install Railway CLI if needed
./install_railway.sh             # macOS/Linux
.\install_railway.ps1            # Windows

# Link to project
railway link -p 0e58b112-f5f5-4285-ad1f-f47d1481045b

# Deploy
./deploy-secure.sh
